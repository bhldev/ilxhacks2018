To start service
================

1. modify line 5 in app.js to port 8080 (or 10000)

2. modify line 3 in index.html to your IP address plus port (ifconfig)

3. node app.js

4. look in browser ex http://10.10.121.7:8080

5 look in console for hello world

To start website
================

1. cd laterunners

2. yarn start
